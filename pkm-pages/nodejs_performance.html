  • https://clinicjs.org/bubbleprof/ is an async-aware profiler that can help find the real bottlenecks in applications.
  • Node.js now has the v8.getHeapSnapshot function which returns a ReadableStream of the snapshot data. This data can be inspected by the Chrome dev tools to check memory usage, leaks, etc.
