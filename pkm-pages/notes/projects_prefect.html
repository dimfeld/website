--- title: "Prefect" tags: date: 2022-10-02 updated: 2022-10-02 ---
  • status: Early
  • url: https://github.com/dimfeld/prefect
  • A task queue based on SQLite, which will eventually be usable either embedded as a library or standalone.
  • Features
    • Scheduled Tasks
    • Repeating Tasks
    • Multiple types of tasks
    • Task can checkpoint and update payload, so that retries will start from there.
    • Tasks can submit a heartbeat while they are still working on long tasks.
    • Tasks expire and are retried if the heartbeat times out for too long
    • Worker Tasks
      • can listen only for specific types of tasks.
      • can call a function to wait for the next task, which the library will resolve once one is available.
  • In the future this library will be runnable standalone, with task clients connecting over HTTP or gRPC.
    • HTTP Long polling and gRPC streaming to wait for task availability
